# DevTool

A lightweight Windows terminal dashboard built with Batch scripting.

DevTool provides a fast command-line interface for launching:
- Applications
- Development tools
- Websites
- Project boards
- Study resources
- Custom shortcuts

Designed for developers, students, and power users who want a minimal and fast productivity hub directly from the Windows terminal.

## Features
- Colorful terminal UI
- Menu-based navigation
- Quick-launch applications
- Website shortcuts
- Customizable menus
- Lightweight and portable
- Pure Batch scripting

## Technologies
- Windows Batch (.bat)
- CMD ANSI colors
- Windows shortcut integration

## Customization
You can easily:
- Replace links
- Add your own tools
- Create new menus
- Personalize colors and layout
- *Make sure the programs you add are in the /files/ folder relative to the program itself*
